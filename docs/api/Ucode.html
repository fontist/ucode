ucode — Unicode Character Database toolkit.

Top-level hub. Every namespace under Ucode has its own hub file at lib/ucode/.rb declaring autoloads for its children. This file autoloads those hubs plus the flat Ucode::* classes. Files are loaded lazily on first reference.
