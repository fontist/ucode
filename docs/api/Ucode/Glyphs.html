Overview

Glyphs — converts Code Charts PDF pages into per-codepoint SVGs.

Pipeline: fetch per-block PDF → render to SVG → detect grid → extract cell → normalize viewBox → write glyph.svg.

Vector extraction only. NEVER run OCR.
