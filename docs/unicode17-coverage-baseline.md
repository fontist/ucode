# Unicode 17.0 baseline coverage audit

Status: **partial** — cmap-verified for 5 blocks (Sidetic, Beria Erfe, CJK Ext
H, CJK Ext J, Adlam); publisher-claimed for the remaining blocks pending Tier 1
font acquisition.

This report grounds every later decision about the canonical 4-tier resolver
(TODO 20) and the Unicode 17 dataset build (TODO 21): which blocks are already
fully covered by an obtainable Tier 1 font, which need Pillar 1/2 (PDF-embedded
font extraction), and which need Pillar 3 (Last Resort placeholders).

## Methodology

For each Unicode 17 block:

1. **Tier 1 (real-font cmap)** — load the publisher-confirmed font, walk its
   `cmap` table, intersect with the block's assigned codepoints per UCD 17
   `Blocks.txt` + `UnicodeData.txt`.
2. **Pillar 1 (PDF-embedded CIDFont)** — for fonts embedded inside the
   per-block Code Charts PDF, walk the Type0 → CIDFont → FontDescriptor →
   FontFile2/3 object graph and use `/ToUnicode` to map CID → codepoint.
3. **Pillar 2 (content-stream correlation)** — for embedded CIDFonts without
   `/ToUnicode`, render the page to SVG, partition labels from specimens, and
   match positionally. Proven on Tai Yo; Sidetic + Beria Erfe are the
   generalization targets.
4. **Pillar 3 (Last Resort UFO)** — placeholder glyphs for unassigned, PUA,
   noncharacter, and any codepoint no higher tier produced.

**Verification status** (per row):

- ✅ **cmap-verified** — Tier 1 cmap audit completed; numbers are from the
  font's actual cmap, not publisher claims.
- 🟡 **publisher-claimed** — publisher documentation says the font covers the
  block; cmap audit not yet run (font not locally available).
- ⬜ **pending** — no Tier 1 font identified yet; will need Pillar 1/2/3.

## Procedure to complete this audit

For each 🟡 / ⬜ block:

1. **Acquire the font.** See the Tier 1 source column. FSung is already local
   at `~/Downloads/全宋體/`; NotoSansAdlam is in `spec/fixtures/fonts/`.
2. **Run the audit** (post-PR-C8 CLI):
   ```
   bundle exec exe/ucode audit font <path> \
     --label <font-label> \
     --unicode-version 17.0.0 \
     --output ./output
   ```
   Output lands at `output/font_audit/<label>/index.json`.
3. **Extract the block row** from `block_summaries` and update the summary
   table in this doc.
4. **For pillar 2 verification** (Sidetic, Beria Erfe only):
   ```
   bundle exec exe/ucode glyphs --block <block-first-cp> --version 17.0.0
   ```
   The `Catalog` reports `#codepoints`, `#size`, `#font_count` per block.
5. **Cross-check**: Tier 1 cmap count + pillar 1-2 chart count should equal
   `total_assigned` for that block.
