Overview

A NamesList.txt entry header plus its annotations. Each annotation array holds typed Relationship subclass instances (see Relationship).

The Coordinator flattens these into CodePoint.relationships for the primary codepoint; this standalone model is emitted only when a consumer needs the raw, scope-preserved grouping.
